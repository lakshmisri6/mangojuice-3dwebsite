# Mango Juice Website 🥭

A modern and interactive 3D product showcase website for Mango Juice, built with Next.js, TypeScript, and Tailwind CSS.

## 🚀 Features

* Modern and responsive UI
* 3D product showcase
* Smooth scrolling animations
* Mobile-friendly design
* Fast performance with Next.js
* Clean and reusable component structure

## 🛠️ Tech Stack

* Next.js 14
* React
* TypeScript
* Tailwind CSS
* Vercel (Deployment)

## 📂 Project Structure

```text
app/
components/
data/
public/
```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/lakshmisri6/mangojuice.git
```

Navigate to the project folder:

```bash
cd mangojuice
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
